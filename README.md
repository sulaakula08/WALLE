# WALLE · Walley ♻️

**Умная переработка / Ақылды қайта өңдеу**

Мобильное приложение (Expo / React Native), которое объединяет:

- 🗺️ **Карту пунктов приёма** — ближайшие точки сортировки
- 🤖 **AI-распознавание отходов** — камера + Gemini Vision определяет тип материала
- 🦾 **Вызов робота** — робот забирает отходы, взвешивает, присваивает ID и везёт на сортировку
- 📇 **Цифровой паспорт** — путь отходов в реальном времени: от передачи до подтверждения переработчиком
- 🧠 **Эко-квизы** — вопросы генерирует Gemini
- 🏆 **Геймификация и кланы** — уровни, достижения, рейтинг

Интерфейс на русском и казахском языках.

## Запуск

```bash
npm install
npx expo start
```

Откройте проект в **Expo Go** на Android-телефоне (QR-код в терминале).

> ⚙️ Ключи API (Gemini, Google Maps) вынесены в `src/config.js` и `app.json`.
> Не коммитьте реальные ключи в публичный репозиторий — используйте переменные окружения.
